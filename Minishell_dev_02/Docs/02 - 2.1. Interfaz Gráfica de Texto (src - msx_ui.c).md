### **2.1. Interfaz Gráfica de Texto (`src/msx_ui.c`)**

**Objetivo de este archivo:**
Aunque Minishell es una interfaz de línea de comandos (CLI), tiene una "cara" visible: lo que el usuario ve antes de escribir y cómo se le pide información. En tu código, has optado por una estética retro inspirada en el sistema **MSX BASIC**.

Este archivo agrupará todas las funciones que imprimen mensajes de bienvenida o gestionan el *prompt* (el texto ` MSX>  `). Separar esto del bucle principal (`loop.c`) mantiene el código limpio y modular: si mañana quieres cambiar la estética a "Matrix", solo tendrás que tocar este archivo.

**Instrucción:**
Crea el archivo `src/msx_ui.c`. De momento, solo necesitamos incluir nuestra cabecera general para tener acceso a `printf`, `readline` y las definiciones estándar.

Copia este código inicial en `src/msx_ui.c`:

```c
#include "minishell.h"
```
