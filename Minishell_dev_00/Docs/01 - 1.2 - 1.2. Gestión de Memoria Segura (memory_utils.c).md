### **1.2. Gestión de Memoria Segura (`src/utils/memory_utils.c`)**

**Objetivo de este archivo:**
En C, gestionar la memoria manualmente es arriesgado. Si `malloc` falla (devuelve `NULL`) y no lo comprobamos, el programa puede romperse (Segfault) en cualquier momento. Para no escribir esa comprobación cientos de veces, crearemos este archivo de utilidades que centraliza la gestión segura.

**Instrucción:**
Crea el archivo `src/utils/memory_utils.c` y añade únicamente la cabecera necesaria para que reconozca nuestras estructuras y funciones de error.

```c
#include "minishell.h"
```
