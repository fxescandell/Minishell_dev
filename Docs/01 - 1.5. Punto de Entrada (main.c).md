### **1.5. Punto de Entrada (`src/main.c`)**

**Objetivo de este archivo:**
Este es el "jefe" de tu programa. Su única responsabilidad es orquestar el flujo de vida de la shell:

1.  **Nacer:** Preparar las estructuras y el entorno (`init`).
2.  **Vivir:** Entrar en el bucle infinito de lectura y ejecución (`loop`).
3.  **Morir:** Limpiar todo y salir ordenadamente (`cleanup`).

**Instrucción:**
Crea el archivo `src/main.c`. De momento, solo incluiremos nuestra cabecera general para tener acceso a todas las definiciones que hemos creado (structs, funciones de error, etc.).

Copia este código inicial en `src/main.c`:

```c
#include "minishell.h"
```
